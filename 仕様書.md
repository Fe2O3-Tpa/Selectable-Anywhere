# Selectable Anywhere v2.0
## プロジェクト概要
- プロジェクト名: Selectable Anywhere v2.0
- 目的: 選択(コピー)不可テキストを選択可能にする
- 対象ユーザー: Web開発者、ユーザースクリプト利用者
- 利用シーン: コピー制限されたWebページでテキストを選択し、取得できるようにする
## 開発環境
- プログラミング言語：JavaScript（ES2020相当）
- フレームワーク / ライブラリ：Tampermonkey（GM_getValue / GM_setValue）
- 開発ツール（IDE、エディタ等）：Visual Studio Code、Codex、Chromium DevTools
- バージョン管理システム：Git / GitHub
## 技術仕様
- 実行環境:
  - Tampermonkey UserScript（`@run-at document-start`、`@noframes`）
- 状態管理:
  - `enabled_${location.origin}` を接頭辞に `enabled` / `uiFolded` を保存・読込
- 選択制限解除:
  - `getComputedStyle(...).userSelect === "none"` の要素に対して `user-select: text !important` を付与
  - 変更前のインライン値（value / priority）を `WeakMap` に保存し、OFF時に復元
- 監視:
  - `MutationObserver` で `style` 属性変更と追加ノードを監視
  - 監視対象は `document.documentElement`、`subtree: true`
  - 再評価は 50ms デバウンスで実行
- イベント妨害解除:
  - 対象イベントは `selectstart`, `mousedown`, `pointerdown`, `contextmenu`, `copy`
  - `document` に capture フェーズで登録し、`stopImmediatePropagation()` で遮断
  - OFF時は `removeEventListener` で全解除
- 除外ポリシー:
  - `input` / `textarea` / `button` / `select` / `option` / `canvas` / `svg` / `video` / `audio` / `contenteditable` / `[data-sa-ui]` は処理対象外
  - Shadow DOM 内要素は対象外
## 機能・構造の振り返り
1. 選択制限解除
   目的: コピー不可テキストを選択可能にする
   動作: 計算後スタイルが user-select:none の要素に対し user-select:text !important を適用
   使用技術:
   - JavaScript
   - Tampermonkey
   - getComputedStyle

2. 安全な復元処理
   目的: 無効化時にページ側スタイルへ安全に戻す
   動作: 変更前のインライン user-select 値と priority を保存し、OFF時に保存値へ復元
   使用技術:
   - JavaScript
   - WeakMap
   - CSSOM（style.setProperty / removeProperty）

3. 変更追従監視
   目的: 動的に追加・更新された要素にも解除処理を適用する
   動作: MutationObserver で style 属性変更と追加ノードを監視し、変更点のみ再評価
   使用技術:
   - JavaScript
   - MutationObserver
   - Set / WeakRef

4. JSイベント妨害解除
   目的: ページ側の選択・コピー妨害ハンドラを最小干渉で無効化する
   動作: selectstart/mousedown/pointerdown/contextmenu/copy を capture で受け、stopImmediatePropagation() で遮断
   使用技術:
   - JavaScript
   - addEventListener / removeEventListener（capture）
   - Map / Set

5. ON/OFFトグルUI
   目的: ページ上から即時に有効/無効を切り替える
   動作: 右下固定UIクリックで状態を反転し、機能適用/復元を切り替え（Shift+クリックでUI折りたたみ）
   使用技術:
   - JavaScript
   - DOM API
   - CSS（インラインスタイル）

6. 状態永続化
   目的: サイトごとに前回状態を維持する
   動作: origin単位キーで enabled / uiFolded を保存・読込
   使用技術:
   - Tampermonkey GM_getValue / GM_setValue
   - location.origin

## デバッグログ一覧

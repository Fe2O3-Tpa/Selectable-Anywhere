# Selectable Anywhere v1.5
## プロジェクト概要
- プロジェクト名: Selectable Anywhere v1.5
- 目的: 選択(コピー)不可テキストを選択可能にする
- 対象ユーザー: Web開発者、ユーザースクリプト利用者
- 利用シーン: コピー制限されたWebページでテキストを選択し、取得できるようにする
## 機能・構造の振り返り
1. 選択制限解除
   目的: コピー不可テキストを選択可能にする
   動作: 計算後スタイルが user-select:none の要素に対し user-select:text !important を適用
   使用技術:
   - JavaScript
   - Tampermonkey
   - getComputedStyle

2. 安全な復元処理
   目的: 無効化時にページ側スタイルへ安全に戻す
   動作: 変更前のインライン user-select 値と priority を保存し、OFF時に保存値へ復元
   使用技術:
   - JavaScript
   - WeakMap
   - CSSOM（style.setProperty / removeProperty）

3. 変更追従監視
   目的: 動的に追加・更新された要素にも解除処理を適用する
   動作: MutationObserver で style 属性変更と追加ノードを監視し、変更点のみ再評価
   使用技術:
   - JavaScript
   - MutationObserver
   - Set / WeakRef

4. ON/OFFトグルUI
   目的: ページ上から即時に有効/無効を切り替える
   動作: 右下固定UIクリックで状態を反転し、機能適用/復元を切り替え
   使用技術:
   - JavaScript
   - DOM API
   - CSS（インラインスタイル）

5. 状態永続化
   目的: サイトごとに前回状態を維持する
   動作: origin単位キーで enabled / uiFolded を保存・読込
   使用技術:
   - Tampermonkey GM_getValue / GM_setValue
   - location.origin

## デバッグログ一覧
- 有効化スイッチ:
  - `DEBUG` 変数で制御（`true` でログ出力、`false` で無効）
- 出力関数:
  - `log(message)` 関数を使用
  - 出力形式は常に `"[Selectable Anywhere] " + message`

1. 状態切替ログ
   - ログ: `true` / `false`
   - 出力箇所: `reevaluateAll()`
   - 用途: 現在の有効状態（ON/OFF）確認

2. ON開始ログ
   - ログ: `enable cycle started: trackedRefs={数値}`
   - 出力箇所: `applySelectablePolicyOnEnable()`
   - 用途: ONサイクル開始時の追跡要素数を確認

3. 初回走査ログ
   - ログ: `initial scan capped: processed={数値}, total={数値}`
   - ログ: `initial scan full: processed={数値}`
   - 出力箇所: `scanWholeDocument()`
   - 用途: 初回走査が上限到達したかを確認

4. 復元発生ログ（ON中）
   - ログ: `restore happened while enabled`
   - 出力箇所: `restoreInlineUserSelect()`
   - 用途: ON中に復元が発生していないかを確認

5. 監視フラッシュ要約ログ
   - ログ: `flush summary: roots={数値}, processed={数値}, attrMut={数値}, childMut={数値}`
   - 出力箇所: `flushPending()`
   - 用途: MutationObserver入力量と処理量の相関確認

6. OFF時サマリログ
   - ログ: `disable summary: forced={数値}, restored={数値}, restoredWhileEnabled={数値}, observedNone={数値}, mutationBatches={数値}, attrMut={数値}, childMut={数値}, scan={数値}/{数値}, trackedRefs={数値}, uptimeMs={数値}`
   - 出力箇所: `restoreSelectablePolicyOnDisable()`
   - 用途: 1サイクル全体の統計確認（強制適用・復元・監視・走査・稼働時間）
